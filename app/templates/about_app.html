A propos de GrandPy Bot:

A propos de l'application

Cette application a été créée en réponse au projet n°7 du Parcours Développeur d'Application Python d'OpenClassrooms. Le but étant de déployer une première WebbApp "complète".
Pour plus d'informations, vous retrouverez le sujet ici.

Les Technologies

Elle a été développé avec HTML5/CSS3 avec Bootstrap. La partie front-end est gérée avec JavaScript (jQuery) et la partie back-end avec Python (plus précisément Flask).
Les versions ont été gérées via Git (dont le lien du dépôt est disponible en bas de la page), et la planification par un tableau Trello. ENfin
